# IPO

Access IPO data.

## `client.ipo.current(as_df=False)`

Currently open IPOs.

```python
data = client.ipo.current()
df = client.ipo.current(as_df=True)
```

## `client.ipo.upcoming(as_df=False)`

Upcoming IPOs.

```python
data = client.ipo.upcoming()
df = client.ipo.upcoming(as_df=True)
```
